## SeatScope
SeatScope is a concert ticket website I built using HTML, CSS, and JavaScript.
It lets people search for concerts, pick seats, add tickets to a cart, and go through a fake checkout process.
I made this project to practice building a real-world type website without using any frameworks.
## What It Can Do
 Search for artists or venues
 Filter by genre (Hip-Hop, Pop, R&B, EDM)
 Sort by price or demand
 See countdown timers for upcoming events
 Save favorite events
 Pick actual seats (VIP, Standard, Budget)
 Add tickets to a cart
 Increase or decrease amount
Fill out a form
Switch between dark mode and light
## What I Used
HTML
CSS
JavaScript (no libraries)
I didn’t use React or anything like that. Everything is built with regular JavaScript and DOM manipulation.
